# 鲁班H5 v.1.8.0 至 v.1.8.1 迁移指南

升级鲁班H5版本至 v.1.8.1.

主要修改是：

  • 将 strapi依赖从 beta.17.x 升级至 beta.18.4， 主要为了修复 图片上传之后，返回的response url 中包含 localhost:1337

  • 更多请参见 Strapi 升级指南: Strapi Migration guide from beta.17+ to beta.18

将 鲁班H5 升级到v.1.8.1 比较简单，只要执行如下命令即可：

# clone 最新代码
# 删除后端相关的缓存、admin后台、node_modules
# !! 注意下面的命令会先停止服务
./luban-h5 clean
# 重新安装依赖，构建admin后台
./luban-h5 init
./luban-h5 start

如果有疑问，提 issue 即可
